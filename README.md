# pong

A simple retro Pong game (HTML5 Canvas, no dependencies).

### Run

- **Fastest**: open `index.html` in your browser.
- **Recommended** (avoids some browser file restrictions):

```bash
cd /Users/tungnguyen/Documents/GitHub/pong
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

### Controls

- **Serve / start**: Space (or click/tap the game)
- **Move**: W/S or ↑/↓
- **Pause**: P (only while ball is in play)
- **Restart**: R

### Rules

- **First to 11** wins.
